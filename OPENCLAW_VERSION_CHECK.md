## 版本对比

### 当前版本：2026.2.26
- 发布日期：2026年2月26日
- 状态：已安装
- 安全性：✅ 已修复 ClawJacked 漏洞

### 最新版本：2026.3.2
- 发布日期：2026年3月3日（5天前）
- 状态：⚠️ 未安装
- 主要更新：
  - PDF 分析工具增强
  - Telegram 流式传输默认开启
  - MiniMax 模型支持
  - 会话附件支持
  - 插件 SDK 改进
  - 多项安全修复

## 自动更新设置

**当前状态**：未配置

**建议配置**：
```bash
# 开启自动更新检查
openclaw config set update.auto true

# 设置更新通道（stable 或 beta）
openclaw config set update.channel stable

# 配置更新检查间隔（每天）
openclaw config set update.checkInterval 86400
```

## 升级建议

### 方案A：手动升级（推荐）
```bash
# 使用 npm 升级
npm update -g openclaw

# 或重新安装最新版
npm install -g openclaw@latest
```

### 方案B：保持当前版本
- 当前版本 2026.2.26 已修复关键安全漏洞
- 如不需要新功能，可暂缓升级
- 建议关注官方安全公告
