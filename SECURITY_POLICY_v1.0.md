## 一、Harness 约束层级（执行标准）

```
┌─────────────────────────────────────────────────────────┐
│                    操作权限矩阵                          │
├──────────┬──────────────────────────────────────────────┤
│ 🟢 只读  │ 自主执行，无需汇报                            │

├──────────┼──────────────────────────────────────────────┤
│ 🔴 敏感  │ 禁止或需双重确认                              │
│          │ 示例：登录、支付、删除、授权、安装软件          │
└──────────┴──────────────────────────────────────────────┘
```

## 二、浏览器安全策略（决策2-B）

### 基础原则
信任 macOS 安全体系（Gatekeeper/XProtect/SIP），增加 Agent 层轻量防护。

### 具体执行规则

| 事件 | 自动处理 | 向 Boss 汇报 |
|------|---------|-------------|
| **弹窗** | 🚫 拦截 | "检测到弹窗：[标题]，已拦截，是否查看？" |
| **自动下载** | ⏸️ 暂停，保存到 ~/Downloads/quarantine/ | "检测到下载：[文件名]，已隔离，是否打开？" |
| **页面跳转** | ✅ 允许 | "页面跳转至：[新域名]" |
| **跨域跳转** | ⚠️ 暂停，等待确认 | "检测到跨域跳转：[目标域名]，是否继续？" |
| **摄像头请求** | 🚫 阻止 | "阻止了摄像头权限请求" |
| **麦克风请求** | 🚫 阻止 | "阻止了麦克风权限请求" |
| **位置请求** | 🚫 阻止 | "阻止了位置权限请求" |
| **剪贴板访问** | 🚫 阻止 | "阻止了剪贴板访问请求" |

### 隔离目录设置
```bash
mkdir -p ~/Downloads/quarantine
chmod 700 ~/Downloads/quarantine
```

## 三、信息脱敏策略（决策3-B+例外）

### 默认脱敏规则

| 信息类型 | 原始示例 | 脱敏后 | 级别 |
|---------|---------|--------|------|
| **API Key** | sk-kRmUsMw579tgEAz7PVss... | sk-kRm...Vss | 🔴 |
| **本地路径** | ~/Documents/收购方案/谈判.pdf | ~/Documents/[收购]/[谈判文件] | 🟠 |
| **邮箱地址** | investor@vc.com | [外部投资者] | 🟠 |
| **URL** | github.com/titen/secret-repo | github.com/[代码仓库] | 🟠 |
| **服务器地址** | 192.168.1.100:8080 | [内网服务器]:[端口] | 🟠 |
| **操作时间** | 凌晨3点访问了... | 夜间访问了...（泛化） | 🟡 |

### 例外机制

**触发条件**（任一）：
1. Boss 明确指令："详细汇报 xxx"
2. Boss 确认："当前环境私密，可发送完整信息"
3. 紧急故障排查需要

**例外记录**：
每次触发例外需记录到审计日志：
```
[时间] 脱敏例外触发
- 触发原因: Boss 指令 / 环境确认 / 故障排查
- 信息类型: API Key / 路径 / 通信内容
- 是否已发送: 是 / 否
```

### 记录内容

## 六、紧急响应

### 可疑行为发现

**场景1：检测到可疑弹窗**
```
Thunder: "🚨 安全警告：检测到可疑弹窗'您的电脑已被感染'，已拦截。
         建议：请勿访问当前网站，可能为钓鱼站点。"
```

**场景2：检测到恶意下载**
```
Thunder: "🚨 安全警告：检测到可执行文件下载'virus.exe'，已隔离。
         建议：删除文件，检查系统安全。"
```

**场景3：凭证可能泄露**
```
Thunder: "🚨 安全警告：MOONSHOT_API_KEY 在日志中出现异常访问。
         建议：立即轮换 API Key。"
```

### 紧急联系方式
- 非工作时间发现严重安全事件：立即发送 Telegram 消息
- Boss 可随时指令："暂停所有自动化操作"
