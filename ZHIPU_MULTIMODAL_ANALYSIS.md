## 三、各场景模型选择指南

### 场景1：图片 + 文本分析
```
任务: "分析这张图表并总结趋势"
模型: GLM-4.6V 或 GLM-4V
流程: 上传图片 → 模型理解 → 文本输出
价格: 按输入图片大小 + 输出文本计费
```

### 场景2：文生图
```
任务: "生成一张未来城市的图片"
模型: CogView-3
流程: 文本提示 → 生成图片 → 返回URL
价格: ~0.1-0.3元/张
```

### 场景3：代码生成
```
任务: "写一个Python爬虫抓取新闻"
模型: GLM-5-Code（首选）或 GLM-5
优势: 工程级代码，支持复杂项目结构
价格: 6-8元/百万tokens
```

### 场景4：视频生成
```
任务: "生成一段风景视频"
产品: 清影
注意: 可能需要通过App或特定API调用
价格: 按需付费
限制: 目前API开放程度需确认
```

### 场景5：全能型任务
```
任务: "分析这份PDF报告，提取关键数据，生成图表，并写总结"
模型组合:
  1. GLM-4-Long（处理PDF长文本）
  2. GLM-4.6V（理解图表图片）
  3. CogView-3（生成新图表，如需要）
  4. GLM-5（写总结报告）
```

## 四、关键问题确认

### 需要向智谱确认的事项：

1. **视频API开放程度**
   - 清影是否完全开放API？
   - 价格如何计算？
   - 有无免费额度？

2. **免费模型限制**
   - GLM-4-Flash 免费额度具体是多少？
   - 是每月免费还是永久免费？
   - 速率限制（RPM/QPM）是多少？

3. **多模态调用方式**
   - 图片输入是传URL还是base64？
   - 图片输出是返回URL还是直接下载？
   - 是否支持批量处理？

4. **CogView-3 定价**
   - 精确价格是多少/张？
   - 有无免费额度？
   - 图片分辨率限制？

## 五、与现有 Moonshot 的协作方案

### 双模型分工建议

| 任务类型 | 推荐模型 | 理由 |
|----------|----------|------|
| 中文内容理解 | Moonshot | 已验证，响应稳定 |
| 代码生成 | GLM-5-Code | 专业代码模型，SOTA |
| 图片分析 | GLM-4.6V | 智谱视觉模型更强 |
| 图片生成 | CogView-3 | Moonshot 无此能力 |
| 超长文本(>200K) | GLM-4-Long | 支持1M上下文 |
| 复杂推理 | GLM-Z1系列 | 专门优化推理 |
| 视频生成 | 清影 | Moonshot 无此能力 |
| Agent任务 | GLM-5 | AutoGLM生态支持 |
